# TabMark2 二次开发版

> 本仓库为基于原项目进行功能增强与体验优化的二次开发版本。

## 原项目说明

- 原项目名称：`TabMark-Bookmark New Tab`
- 原仓库地址：`https://github.com/Alanrk/TabMark-Bookmark-New-Tab`
- 感谢原作者提供的产品思路与基础实现

## 当前仓库定位

这个版本基于原项目继续迭代，主要目标是：

- 优化主页交互体验
- 增强快捷标签能力
- 增强二级文件夹标签能力
- 提升页面可维护性与性能
- 为后续跨设备备份 / 恢复等能力做准备

## 已完成的二开功能

### 主页书签体验
- 将二级文件夹改造成顶部可切换标签
- 默认增加“全部”标签，展示直属书签
- 支持点击不同二级标签切换下方书签内容
- 支持主页直接拖拽调整二级标签顺序
- 二级标签顺序会持久化保存

### 快捷标签体验
- 中间快捷标签改成纯固定模式
- 不再根据访问频率和历史记录自动变化
- 支持主页直接添加快捷标签
- 支持设置页添加 / 编辑 / 删除快捷标签
- 支持主页直接拖拽快捷标签排序
- 支持设置页拖拽排序
- 增加数量上限控制，避免快捷标签过多影响布局

### 左侧边栏搜索体验
- 左侧边栏改为搜索优先模式
- 支持全局检索所有书签与文件夹
- 支持中文 / 英文别名匹配
- 支持键盘上下选择、Enter 打开和鼠标滚轮滚动结果

### 交互与 UI 优化
- 优化主页添加快捷标签入口样式
- 保证主页始终保留添加快捷标签入口
- 修复顶部标签右键后文字异常高亮问题
- 优化部分布局抖动问题

### 代码整理与性能优化
- 清理一批未接入文件、无效依赖和高置信废代码
- 合并部分重复函数与重复初始化逻辑
- 优化部分背景色采样和历史查询热点逻辑
- 为后续 WebDAV 备份与恢复功能整理数据结构

## 当前主要功能

- 将书签目录作为新标签页主页展示
- 支持侧边栏书签浏览
- 支持悬浮球快速访问
- 支持 AI 搜索与多搜索引擎切换
- 支持固定快捷标签
- 支持自定义主题与壁纸
- 支持书签拖拽排序
